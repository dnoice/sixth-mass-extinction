## Key Contributions by Paper

### Core E/MSY Methodology
- **Ceballos 2015 [S2]**: Establishes 100x extinction rate using conservative E/MSY calculation
- **Pimm 2014 [S49]**: Comprehensive review supporting ~0.1 E/MSY background rate
- **De Vos 2015 [S50]**: Methodological deep-dive proving 0.1 E/MSY baseline (via Duke repository)

### Mass Extinction Context
- **Barnosky 2011 [S48]**: Deep-time comparison; 75% threshold for mass extinction
- **Ceballos 2023 [S53]**: Genus-level analysis showing 35x acceleration

### Synthesis
- **IPBES 2019 [S1]**: Authoritative ~1 million species at risk figure
- **Ceballos 2017 [S3]**: Population-level "biological annihilation" beyond species extinctions

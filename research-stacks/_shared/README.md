# Shared Resources

## Cross-Section Resources for Research Stacks

> **Purpose:** Common datasets, utilities, and styles used across multiple episodes.

---

## Directory Structure

```tree
_shared/
├── data/           # Common datasets (IUCN, LPI, etc.)
├── utils/          # Shared Python utilities
└── styles/         # Visualization style configs
```

---

## Shared Data

Common datasets that appear in multiple episodes:

| Dataset | Description | Source |
| ------- | ----------- | ------ |
| IUCN Red List | Species threat assessments | iucnredlist.org |
| Living Planet Index | Population trends | livingplanetindex.org |
| IPBES Assessments | Biodiversity summaries | ipbes.net |
| FAO STAT | Agriculture/forestry data | fao.org |

---

## Utilities

Python modules for common operations:

| Module | Purpose |
| ------ | ------- |
| `data_utils.py` | Data loading, cleaning functions |
| `viz_utils.py` | Standard visualization functions |
| `stats_utils.py` | Statistical calculations (E/MSY, etc.) |

---

## Styles

Visualization configuration for consistency:

| File | Purpose |
| ---- | ------- |
| `sme_style.mplstyle` | Matplotlib style config |
| `plotly_template.json` | Plotly theme config |
| `color_palettes.py` | Standard color schemes |

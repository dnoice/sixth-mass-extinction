## FILE NAMING CONVENTION - DETAILED BREAKDOWN

All research blueprint files follow a systematic naming pattern to clearly identify their position in the Master Blueprint hierarchy. Understanding these abbreviations is critical for navigation and future development.

### **Abbreviation Key:**

| Abbreviation | Full Name | Meaning | Examples |
|--------------|-----------|---------|----------|
| **MB** | **MASTER_BLUEPRINT** | Links to the main MASTER_BLUEPRINT.md framework document | All files reference MB |
| **PS** | **PARENT_SECTION** | Main sections of the Master Blueprint (10 total) | 1.0, 2.0, 3.0 ... 10.0 |
| **SS** | **SUB-SECTION** | Detailed subsections under each Parent Section | 1.1, 1.2, 2.1, 2.2, etc. |

### **File Naming Patterns:**

#### **Pattern 1: Parent Section Blueprints**

```
Format:  RESEARCH_BLUEPRINT_MB_PS-X.0_SECTION_NAME.md
Example: RESEARCH_BLUEPRINT_MB_PS-2.0_PRIMARY_DRIVERS.md

Breakdown:
- RESEARCH_BLUEPRINT = File type identifier
- MB = Links to MASTER_BLUEPRINT
- PS = This is a Parent Section
- 2.0 = Section number (one of 10 main sections)
- PRIMARY_DRIVERS = Descriptive section name
- .md = Markdown file extension
```

**What it means:** This file contains the overview and framework for an entire main section (e.g., Section 2.0: Primary Drivers of Extinction), typically 10-15 pages outlining all sub-sections within it.

#### **Pattern 2: Sub-Section Blueprints**

```
Format:  RESEARCH_BLUEPRINT_MB_SS-X.Y_SUBSECTION_NAME.md
Example: RESEARCH_BLUEPRINT_MB_SS-2.3_CLIMATE_CHANGE.md

Breakdown:
- RESEARCH_BLUEPRINT = File type identifier
- MB = Links to MASTER_BLUEPRINT
- SS = This is a Sub-Section
- 2.3 = Sub-section number (Section 2, Sub-section 3)
- CLIMATE_CHANGE = Descriptive subsection name
- .md = Markdown file extension
```

**What it means:** This file contains a detailed research blueprint for a specific subsection (e.g., 2.3: Climate Change as Extinction Accelerant), typically 30-50 pages with comprehensive research framework, 8 foundational sources, 50-70 supporting papers, methodology, visualizations, and expert contacts.

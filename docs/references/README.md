## Document Descriptions

### Biomes Reference

**File:** `major_minor_biomes.md`
**Coverage:** Complete Earth biome classification including terrestrial (tropical rainforest, temperate forest, boreal, grassland, desert, tundra, Mediterranean, montane) and aquatic (marine, freshwater, wetland, coral reef, deep sea) systems.
**Use For:** Geographic hotspot analysis, habitat destruction research, ecosystem service valuation.

### Animal Groups Reference

**File:** `major_animal_groups.md`
**Coverage:** All major animal taxonomic groups with IUCN threat assessments, population trends, and conservation status. Includes mammals, birds, reptiles, amphibians, fish, invertebrates.
**Use For:** Species-level extinction analysis, taxa-specific threat assessment.

### Plant Groups Reference

**File:** `major_minor_plant_groups.md`
**Coverage:** All major plant taxonomic groups including angiosperms, gymnosperms, ferns, bryophytes. Features 2024 Global Tree Assessment (38% threatened).
**Use For:** Plant conservation, ecosystem foundation analysis, carbon sequestration research.

### Extinction Events Reference

**File:** `major_minor_extinction_events.md`
**Coverage:** All five previous mass extinctions (Ordovician, Devonian, Permian, Triassic, Cretaceous) plus the current Sixth Mass Extinction. Includes causes, species loss metrics, recovery timelines.
**Use For:** Historical comparison, recovery timeframe analysis, extinction dynamics.

### Economic Impacts Reference

**File:** `major_minor_economic_impacts.md`
**Coverage:** Ecosystem service valuation ($125-145T/year), sector-specific impacts, GDP-nature dependencies, financial risk frameworks.
**Use For:** Economic case for conservation, policy analysis, business risk assessment.

### Current Status Reference

**File:** `major_minor_current_status.md`
**Coverage:** Real-time 2024-2025 extinction crisis metrics including IUCN updates, Living Planet Index (73% decline), ongoing extinctions, regional hotspots.
**Use For:** Current state documentation, recent developments, trend tracking.

### Historical Timeline Reference

**File:** `major_minor_historical_timeline.md`
**Coverage:** Human-biodiversity relationship from Late Pleistocene megafauna extinctions through agricultural revolution, colonialism, industrialization to present Anthropocene.
**Use For:** Historical context, human impact trajectory, causal analysis.

### Physical Drivers Reference

**File:** `major_minor_physical_drivers.md`
**Coverage:** Climate physics, ocean acidification, biogeochemical cycles, atmospheric composition, physical tipping points.
**Use For:** Physical science foundations, climate-extinction linkages, threshold analysis.

### Mathematical Models Reference

**File:** `major_minor_mathematical_models.md`
**Coverage:** Quantitative frameworks including E/MSY (extinctions per million species-years), population viability analysis, species-area relationships, network models.
**Use For:** Quantitative analysis, rate calculations, modeling approaches.

## Data Source Standards

All reference documents adhere to tiered source hierarchy:

- **Tier 1:** IPBES, IPCC, FAO, IUCN (authoritative)
- **Tier 2:** Peer-reviewed journals (Nature, Science, PNAS)
- **Tier 3:** Government agencies (EPA, NOAA, national statistics)
- **Tier 4:** NGO reports (WWF, WRI, verified data)
